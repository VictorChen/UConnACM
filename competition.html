ACM Programming Competition

2012 ACM-ICPC North American Qualification Contest

During the ACM programming competition, you will be given a set of problems and your job is to write code to solve them. Once you think your code is correct, you submit it to the website that will be given during the day of the competition. The website will run multiple test cases against your code to determine the correctness and the elapsed time. Even if your code is correct, the website will not accept it if it took too long to run. At the end of the event, the first 3 people who solved the most problems will be declared the winners.

For more information, please see

  • http://dna.engr.uconn.edu/~acm/
  • http://cs.baylor.edu/~hamerly/icpc/qualifier_2012/

You can also contact Professor Ion Mandoiu for details since he is in charge of this event.
